# OpenClaw 总览

OpenClaw 让你把个人 AI 助手运行在自己的机器或服务器上。这一组文档是当前仓库里最适合新用户的主线内容。

<Info>
  这个仓库里同时保留了 `OpenClaw` 当前教程和 `Trellis` legacy 工作流文档。如果你的目标是部署一个
  可长期运行的 AI 助手，从这里开始最合适。
</Info>

## 你会得到什么

- 一个运行在你自己基础设施上的 AI 助手
- 一套包含人格、记忆、Heartbeat 和 Skills 的可扩展系统
- 一条从安装到高级自动化的清晰阅读路径

## 开始前建议准备

- 一台 Linux 服务器或一台长期在线的电脑
- 你计划使用的模型供应商 API Key
- 如果想完全照教程走，建议准备 Telegram
