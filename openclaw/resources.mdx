# OpenClaw resources

Use this page as the companion to the 7-day tutorial. It collects the links you are most likely to need after the first install.

<Info>
  This page belongs to the current maintained `OpenClaw` track. For preserved legacy workflow material, go back to the `Trellis` guide sections instead of treating those pages as the main setup path.
</Info>

## Recommended reading order

1. Start at [OpenClaw overview](./index).
2. Finish [Day 2](./day/2) if you want a working assistant quickly.
3. Read [Day 5](./day/5) before installing third-party Skills.
4. Read [Day 6](./day/6) and [Day 7](./day/7) when you want automation and long-running workflows.

## Common next tasks

- If you want a stable first setup, finish the full 7-day path before adding many Skills.
- If you want better automation, focus on Heartbeat, memory, and a small set of high-value Skills.
- If you want to customize behavior, start with identity files and then layer in tools and scheduled checks.
